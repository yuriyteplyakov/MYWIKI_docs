
# Что такое ООП

> Объектно-ориентированное программирование — это подход к разработке, где программа строится из объектов, которые объединяют данные и методы для работы с ними.

# Основные принципы ООП

**Инкапсуляция**

> объединение данных и методов внутри объекта.

**Наследование**

> возможность создавать новый класс на основе существующего.

**Полиморфизм**

> возможность использовать один интерфейс для разных реализаций.

**Абстракция**

> выделение важных характеристик объекта и скрытие лишних деталей.
