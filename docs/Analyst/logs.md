# Уровни логирования

```
DEBUG
INFO
WARN
ERROR
FATAL
```

**примеры**

> 2026-03-10 10:20:01 INFO Ticket created
> 2026-03-10 10:21:02 ERROR Database timeout

**Как искать ошибку**

> алгоритм диагностики:

1. воспроизвести ошибку
2. посмотреть логи
3. проверить данные в БД
4. проверить интеграции
